clove
===============================

version number: 0.0.1
author: Lamden Team

Overview
--------

Clove is a library that makes atomic swaps between chains easy.

Installation / Usage
--------------------

To install use pip:

    $ pip install clove


Or clone the repo:

    $ git clone https://github.com/Landen/clove.git
    $ python setup.py install
